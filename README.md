## Installation
1. **Clone repository**:
   ```bash
   git clone https://github.com/Reda-Ganoutre01/Ecommerce-Techno-Tech.git
   ```
2. **Setup XAMPP**:
   - Launch XAMPP Control Panel.
   - Start Apache and MySQL.
3. **Database setup**:
   - Access phpMyAdmin: [http://localhost/phpmyadmin](http://localhost/phpmyadmin).
   - Create a database named `ecom_db`.
   - Import the `ecom_db.sql` file.
4. **Configure connection**:
   - Update database connection settings in the project files if necessary.
5. **Access application**:
   - Visit [http://localhost/Ecommerce-Techno-Tech](http://localhost/Ecommerce-Techno-Tech).
