# project_Ecommerce_Techno_Tech

Project web e-commerce pour un magazine gaming.

## Features

### User Features
- Browse and search for products.
- Add products to the cart and favorites.
- Place orders and view order history.
- Responsive design for seamless use on all devices.

### Admin Features
- Manage products (add, update, delete).
- Manage categories and promotions.
- View and manage placed orders.
- Dashboard for analytics and insights.

## Technologies Used

### Frontend
- **HTML5**: Structure of the web pages.
- **CSS3**: Styling and layout.
- **JavaScript**: Interactivity and dynamic content.
- **AOS (Animate on Scroll)**: Smooth animations.
- **Swiper.js**: Product carousels and sliders.

### Backend
- **PHP**: Server-side scripting.
- **MySQL**: Database for storing product, user, and order data.

### Tools
- **XAMPP**: Local development environment.
- **Git**: Version control.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ecommerce-techno-tech.git

2. Move the project to your XAMPP htdocs directory:

```bash
mv ecommerce-techno-tech /C:/xampp/htdocs/
```

3. Start XAMPP and ensure Apache and MySQL are running.

4. Import the database:
- Open phpMyAdmin in your browser (usually at `http://localhost/phpmyadmin`).
- Create a new database named `ecommerce_techno_tech`.
- Import the provided SQL file (`database.sql`) located in the project directory.

5. Configure the project:
- Open the `config.php` file in the project directory.
- Update the database credentials (`host`, `username`, `password`, `database_name`) as per your XAMPP setup.

6. Access the project:
- Open your browser and navigate to `http://localhost/ecommerce-techno-tech`.
